## Local preview

<Steps>
  <Step title="Install Mintlify CLI">
    ```bash
    npm i -g mint
    ```
  </Step>
  <Step title="Run docs locally">
    From the docs folder:

    ```bash
    mint dev
    ```

    Local preview is available at `http://localhost:3000`.

  </Step>
  <Step title="Check for broken links">
    ```bash
    mint broken-links
    ```
  </Step>
</Steps>

## Writing guidelines for this docs set

- Keep scope focused on **Python SDK datasets and evals**
- Prefer runnable snippets that match current SDK APIs
- Use explicit `column_map` in examples for clarity
- Include practical defaults (`workers`, retry, checkpoint, resume)

## Contribution checklist

<AccordionGroup>
  <Accordion title="Before opening a PR" icon="check">
    - Verify new pages are listed in `docs.json` navigation - Validate links
    with `mint broken-links` - Confirm examples align with current `py-sdk` code
  </Accordion>
  <Accordion title="If local preview fails to start" icon="wrench">
    - Reinstall CLI: `npm remove -g mint && npm i -g mint` - Retry on a
    different port: `mint dev --port 3333`
  </Accordion>
</AccordionGroup>
